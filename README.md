## Keyboard Shortcuts

| Keyboard Shortcut | Description                 |
| ----------------- | --------------------------- |
| `q`               | Quits the application       |
| `s`               | Selects the status frame    |
| `b`               | Selects the branches frame  |
| `b`               | Selects the bookmarks frame |
| `l`               | Selects the log frame       |
